# NextNote — Product Roadmap

> Built by an agency owner, for agency owners.

## ✅ Already Built
- Prospect pipeline (New → Contacted → Qualified → Booked → Closed)
- Folder system for organizing leads
- AI-powered spreadsheet import (Claude)
- Appointment booking with Google Calendar + Meet
- Gmail confirmation emails
- AI meeting note summarizer
- Slybroadcast ringless voicemail drops
- AI Insights dashboard
- localStorage persistence
- Appointment history, outcomes, follow-up scheduling
- Reminder banner for upcoming appointments
- Drag & drop MP3 upload for voicemail (in progress)
